# GitHub-Repository-Analyser
GitHub Repository Analyser is a Python script used to fetch and display detailed information about any GitHub repository, this includes stars, forks, open issues, languages used and top contributors.

# Key Features

- **Repository Metrics**:
  - Name, Description
  - Stars, Forks, Open Issues
 
- **Programming Languages**:
  -  List of languages used in the repository

- **Top Contributors**:
  -  Identify the top 5 contributors with contributions count

- **Latest Commit**:
  -  Date of the most recent commit

# Technical Highlights

- Built with Python 3 and `urllib` for API requests
- Interacts directly with GitHub’s REST API
- Handles error states and API rate limits
- Can be extended for automation, analytics, or CI/CD pipelines
